O objetivo desse exercicio eh mostrar como configurar o Webpack para interpretart o ECMAScript2015.

Vale observar que o atributo module foi adicionado ao arquivo webpack.config.js

durante este exercicio.

 Alem disso, os modulos babel-core, babel-loader e babel-preset-es2015 foram instalados usando o comando:

    npm i --save-dev babel-core babel-loader babel-preset-es2015
